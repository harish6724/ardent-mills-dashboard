Shipments
GET /api/shipments List all shipments (paginated, filterable) ▼
Parameters
Response 200
Status Codes
ParameterTypeRequiredDescription
pageintoptionalPage number, default 1
pageSizeintoptionalItems per page, default 10, max 100
statusstringoptionalScheduled | InTransit | Delivered | Delayed
grainTypestringoptionalWheat | Corn | Soy | Oats
originMillIdintoptionalFilter by source mill ID
fromdateoptionalDepartureDate ≥ (yyyy-MM-dd)
todateoptionalDepartureDate ≤ (yyyy-MM-dd)
Example: GET /api/shipments?status=InTransit&grainType=Wheat&page=1&pageSize=10
// PagedResult<ShipmentDto> { "total": 1247, "page": 1, "pageSize": 10, "items": [ { "id": 4821, "shipmentNumber": "SHP-2026-04821", "status": "InTransit", "grainType": "Wheat", "quantityLbs": 46200, "originMillName": "Denver Mill", "originMillId": 1, "destinationCustomer": "Bimbo Bakeries — Dallas, TX", "carrier": "BNSF Railway", "transitMode": "Rail", "departureDate": "2026-06-17", "estimatedArrival": "2026-06-21" } // ... 9 more items ] }
200 OK — paginated list 400 Bad Request — invalid filter params 500 Server Error — unexpected failure
