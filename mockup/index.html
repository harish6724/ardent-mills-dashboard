Architecture Highlights

architecture Key Interview Talking Points

Angular 22 Frontend

bolt
Signals + computed()

Local reactive state without NgRx overhead. ShipmentsStore, MillsStore, QualityStore each hold signal<T[]> + computed() filtered/paged views.

http
httpResource (Angular 19+)

Detail pages use httpResource(() => `/api/shipments/${id()}`) — reactive, auto-cancels on navigation, exposes value(), isLoading(), error().

filter_alt
Three-interceptor pipeline

auth → loading (queueMicrotask guard) → error. Each functional interceptor has one responsibility; fully tree-shakeable.

route
Feature flag guard

CanMatchFn factory — route not evaluated unless flag enabled. Mills and Quality behind flags; Reports disabled (placeholder).

ASP.NET Core 9 Backend

layers
Thin controllers + service layer

Controllers are pure HTTP entry points. All EF Core queries and business logic in IShipmentService, IMillService, etc. — fully testable via mock injection.

schema
DTOs decouple API from DB

ShipmentDto ≠ Shipment entity. API contract stays stable when EF model evolves. Seed data via HasData() in OnModelCreating.

storage
EF Core + LocalDB

Zero-install for demo via SQL Server LocalDB. Same EF Core provider as production Azure SQL — swap connection string only.

api
Swagger / OpenAPI

Self-documenting, browsable API at /swagger. Enables live demo without a frontend build step during interview.
